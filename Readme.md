# 我的研究生论文

希望赶在2015年8月1日之前把初稿完成。

## 工具

- 文本编辑器：Atom

- 中文输入法：搜狗双拼


### Atom配置

- [关闭光标闪烁][1]

  添加以下配置到`~/.atom/style.less`

  ```less
  atom-text-editor::shadow .cursors .cursor {
    opacity: 1;
  }
  ```

- 安装AtomicChar

### 双拼

- 单韵母的字声母部分[用O代替][2]
